# Clima Unificado

Este proyecto guarda la temperatura máxima, mínima y la lluvia diaria en una base de datos `clima.db`.

## Uso

1. Instala las dependencias:
   npm install

2. Ejecuta el script:
   node guardar-clima.js

Se guardarán los datos de los últimos 7 días usando la estación `IALFAR30` y tu API key.
